# Snowcraft 復刻版設計草稿

## 遊戲概述
Snowcraft 是一款經典的雪球大戰即時策略遊戲，玩家控制一隊角色與對手進行雪球戰。

## 核心玩法
- **單人對戰 AI**：玩家（綠隊）vs AI（紅隊）
- **雪球戰鬥**：蓄力投擲雪球攻擊對手
- **角色控制**：滑鼠拖拉移動

## 操作方式
- **移動**：滑鼠拖拉角色
- **投擲**：
  - 按住滑鼠左鍵 → 蓄力（顯示垂直蓄力條）
  - 放開滑鼠左鍵 → 投擲
  - 蓄力越長 → 投擲越遠
  - 蓄力越短 → 投擲越近

## 角色狀態
- 待機 (Idle)
- 移動 (Moving)
- 蓄力中 (Charging)
- 投擲 (Throwing)
- 頭暈 (Stunned) - 被擊中時
- 無敵 (Invincible) - 復活後 1 秒

## 玩家與敵人差異
| 屬性         | 玩家（綠隊）   | 敵人（紅隊）      |
| ------------ | -------------- | ----------------- |
| 人數         | 1 人           | 依關卡（5/10/20） |
| 生命         | 3 條命         | 無                |
| 擊中淘汰次數 | 3 次           | 2 次              |
| 復活         | 有（1 秒無敵） | 無                |

## 遊戲模式
- **單人 vs AI**
- **關卡制**：共 3 關

| 關卡    | 敵人總數 |
| ------- | -------- |
| 第 1 關 | 5 人     |
| 第 2 關 | 10 人    |
| 第 3 關 | 20 人    |

## 數值設定
| 參數                 | 數值            |
| -------------------- | --------------- |
| 玩家淘汰所需擊中次數 | 3 次            |
| 敵人淘汰所需擊中次數 | 2 次            |
| 投擲冷卻時間         | 0.5 秒          |
| 復活無敵時間         | 1 秒            |
| 玩家生命數           | 3 條            |
| 蓄力最大時間         | 0.8 秒          |
| 雪球飛行時間         | 約 1 秒（全場） |
| 頭暈持續時間         | 0.5 秒          |

## 地圖設定
- **比例**：4:3
- **大小**：約可容納 50 位角色
- **佈局**：上方紅隊區域 / 下方綠隊區域

## 敵人 AI
- **投擲模式**：固定時間間隔（0.5 ~ 1 秒）自動投擲
- **目標選擇**：朝玩家方向投擲
- **移動模式**：隨機移動

## UI 顯示
- **玩家狀態**：
  - 剩餘生命數（3 條命）
  - 當前被擊中次數
- **遊戲狀態**：
  - 當前關卡
  - 剩餘敵人數量
- **勝利畫面**：消滅所有敵人時顯示
- **失敗畫面**：玩家 3 條命用盡時顯示
- **蓄力條**：角色旁垂直顯示

## 技術規劃
- **引擎**：原生 Canvas API
- **視角**：俯視角（上方敵方陣營 / 下方我方陣營）
- **美術風格**：簡化幾何圖形（原型階段）
  - 玩家：綠色圓形
  - 敵人：紅色圓形 `#E53935`
  - 雪球：白色圓形（直線飛行）
  - 蓄力條：垂直進度條
